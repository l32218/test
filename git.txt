1、从远程克隆
$ git clone git@github.com:michaelliao/gitskills.git


1、创建分支
$ git checkout -b dev
或者
创建：$ git branch dev 
切换：$ git checkout dev

2、查看分支
$ git branch

3、合并指定分支到当前分支
$ git merge dev
$ git merge --no-ff -m "merge with no-ff" dev

4、删除分支
$ git branch -d dev

5、git log --graph命令可以看到分支合并图

6、工作现场“储藏”
1）储藏
$ git stash
2）查看
$ git stash list
3）恢复
$ git stash apply
$ git stash drop
或者等价于
$ git stash pop
